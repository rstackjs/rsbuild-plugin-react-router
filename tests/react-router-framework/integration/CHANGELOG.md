# integration-tests

## 0.0.0

### Minor Changes

- Unstable Rsbuild support for Node-based Remix apps ([#7590](https://github.com/remix-run/remix/pull/7590))
  - `remix build` -> `rsbuild build`
  - `remix dev` -> `rsbuild dev`

  Other runtimes (e.g. Deno, Cloudflare) not yet supported.
  Custom server (e.g. Express) not yet supported.

  See "Future > Rsbuild" in the Remix Docs for details.
