# React Router 8 Default Template

This example starts as a direct copy of upstream React Router's `integration/helpers/rsbuild-8-template` (collapsed into this repo's corpus as `helpers/rsbuild-template`) and swaps the Rsbuild config/scripts for `rsbuild-plugin-react-router`.

Run `pnpm --filter react-router-8-default-template test:e2e` to exercise development and production browser flows.
